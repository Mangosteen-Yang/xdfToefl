单词突破
单词认读量是关键 看长相认中文即可
不看例句，不看辨析 机械记忆即可
单词一定会读 为听力打基础
单词学习不是阶段性任务 时刻积累单词
词汇书不可能解决所有单词 
单词书背两遍以后
1.阅读文章摘抄生词，摘抄单词、音标、文中含义
2.系统复习摘抄单词，读一篇解决一篇

句子突破
定语后置-》置于名词前
同位语-》与名词合并理解
状语-》结合修饰对象理解
定语从句-》前置OR拆分
同位语从句-》合并OR拆分
状语从句-》结合修饰对象
特殊句式-》倒装还原
  1. 切分句子至简单句单元
  2. 明确主句，并列句，从句
  3. 逐个理解的简单句单元
  4. 安原则处理各从句单元，完成理解

真实信息题
题目特征
  • 信息一定出自原文
  • 只与部分信息相关
题目本质
  • 精确提取+精确理解
核心能力
  • 句子意群的理解能力
解题方法
     找-》读-》选
  • 找：精读题干，提取定位信息以及提问要求；定位不是关键词的重复而是核心概念体系的重复
  • 读：阅读定位信息点周围的信息，通常为上下一两句话（句子理解）
  • 选：正确答案特征：与原文语意一致
题目变化方式
  • 找：定位点的语意对应
  • 读：句法结构更复杂；句子理解信息多
  • 选：迷惑选项干扰
迷惑选项的特征
  1. 无中生有
  2. 偷换概念
  3. 信息杂糅
  4. 答非所问
  • 迷惑选项设置方式：
    • 原文信息+虚假逻辑
    • 原文信息但是无关
    • 原文信息+外来信息
  • 不要凭印象做题
非真实信息题
题目特征
  • 选非真实信息
  • 基于原文真实信息
题目本质
  • 快速获取+精确匹配
核心能力
  • 句子意群的理解能力
解题方法
     找-》读-》选
  • 找：精读题干，明确提问对象；阅读选项，回原文定位选项信息
  • 读：阅读原文信息与选项进行精准匹配；注意：验证信息仍需满足语意的对应性
  • 选：正确答案特征：原文无提及或不符（迷惑选项）
题目变化方式
  • 找：选项顺序与原文顺序不符
  • 读：规格边长，信息分散（两段/全文）=》一次读四个选项，原文顺序找选项
  • 选：四个选项信息在文中均有提及（注意题干）
推理判断题
题目特征
  • 信息强烈暗示但无明确提及
  • 词汇的逻辑性暗示
题目本质
  • 精确理解+逻辑暗示
核心能力
  • 句子意群的理解能力
解题方法
     找-》读-》推-》选
  • 找：阅读题干，明确提问对象，回原文定位相关信息
  • 读：精确理解原文相关信息
  • 推：由原文信息判断推断类型（隐含逻辑关系的提取）
  • 选：正确答案特征：原文信息+原文逻辑转化；注意：不能直接排除含有原文信息的选项
题目变化方式
  • 关系角度：逻辑隐蔽； 引入主观逻辑推理
  • 信息角度：信息理解变难；迷惑选项特征
题目类型
  • 比较逻辑类：比较对象，比较内容，比较结果
  • 取非逻辑类：事物对立取反； 时间点两边取反； 转折关系前后取反
  • 因果逻辑类：隐含因果关系判断， 因果互推
  • 包含关系类：整体包含个体，个体属于整体
  • 细节理解类：逻辑关系隐藏在细节小词之中
指代题
题目特征
  • 确定代词指代对象
题目本质
  • 语言连贯性
核心能力
  • 句间关系判断的能力
解题方法
     找句子-》断关系-》选答案
  • 找句子：前指原则即指代的在前
  • 断关系：根据语意判断关系， 通常在同一意群中
  • 选答案：正确答案：保持前后语意一致性
句子插入题
题目特征
  • 倒数第二题
  • 句子缺失不影响整体理解
题目本质
  • 语意连贯+逻辑连接
核心能力
  • 句子关系判断的能力
解题方法
     读句子-》断关系-》回原文
  • 读句子：以被插入句为核心，精确理解被插入句
  • 断关系：考虑语意和逻辑连接关系，判断3种（上，下，过渡）可能关系
  • 回原文：注意：连贯语意，明确语意不可分
题目变化方式
  • 语意角度：引入干扰信息源
  • 信息角度：句内逻辑关系隐秘
句间连接关系的确定
  • 语意连贯：核心概念会重复出现； 注意：名词的概括， 代词的指代
  • 逻辑衔接：句内谓语动词和状语表示的逻辑衔接； 注意：逻辑连接词
修辞目的题
题目特征
  • 提问信息的目的/关系
  • 服务对象一直在原文
  • 服务关系为核心
题目本质
  • 语意关系+服务关系
核心能力
  • 句见关系判断的能力
解题方法
     读题干回原文-》针对关系提问-》针对目的提问-》正确答案的特征
  • 读题干回原文：判断是针对关系提问还是针对目的提问
  • 针对关系提问：概括信息集合主旨，判断关系； 正确答案：概括主要内容和关系
  • 针对目的提问：阅读原文，通过语意关系判断服务关系； 注意：提示关系词的作用
  • 正确答案的特征：正确答案：概括原文中服务对象； 注意：服务对象由关系确定，不由位置确定
题目变化方式
  • 常考关系确定：
    • 解释：正确答案为解释说明对象（距离， 类比， 标点）
    • 论证（反驳）：正确答案为论证或反驳对象（举例，研究，实验）
    • 总结：正确答案为具体描述特点
    • 强调：正确答案为强调对象（条件关系，转折关系）
  • 选项判定原则： 
    • 选项原文无提及必错
    • 选项既概括原文信息，也不一定正确
词汇题
题目特征
  • 考察单词含义
  • 无明确范围，与理解文章密切相关
题目本质
  • 单词辨识+上下文推理
核心能力
  • 单词的认知能力
解题方法
    考察单词
  • 认识-》选择同义词-》词根词缀
  • 不认识-》上下文推测得出含义-》逻辑关系OR语意连贯性


方法论：如何正确阅读文章

如何有效阅读段落
  1. 理论概述
    1. 段落结构化阅读：结合句间关系，判断段内意群结构
    2. 羽绒服的基本人呢原则：详略得到
  2. 可以略读的内容
    1. 大量数据堆砌
    2. 目的已知的例子与引入
    3. 实验的过程
    4. 重复与进一步解释
    5. 对比，转折前的内容
  3. 句间意群判断方法
     段落意群信息点即为题目考点
托福阅读常见段落结构为
  1. 单观点段落
    1. Structure= Topic Sentence + Details
    2. Structure= Argument + Evidence
    3. Structure= Main point + sub point(s)
    4. Structure= Details+Topic Sentence
    5. Structure= Details + Details
  2. 多观点段落
    1. Structure= Point+transition+Point2
    2. Structure= Point+Point2
如何有效阅读文章
  1. 首段，段意，做段首题
    1. 首段常见情况：General idea Background
    2. 长段落VS短段落：考点分割信息点
  2. 第二段，段内结构，段意，做题
    1. 思维转化一：读主旨求大意不求精确
    2. 思维转化二：做题的过程为信息二次获取过程
  3. 预测下一段
    1. 段间关系：为上文服务VS引出新话题
  4. 验证，段意，依次完成每段题目
    1. 注意：不要跳过段落
  5. 得出文章的结构和主旨
